## 高阶组件
1. 什么是高阶组件
    - 高阶组件就是高阶函数，定义一个函数，立面返回一个有状态组件，就是高阶组件
2. 为什么会出现高阶组件和无状态组件等
    - 将业务逻辑和UI层分离，使得项目环境清晰，便于多人开发和维护
3. 高阶组件分为两种
    1. 属性代理方式
        - 通过属性代理，将包裹的UI组件的props和生成新的props一起传递给UI组件
        - demo02
    2. 反向继承方式
        - 这种方式返回的React组件继承了被传入的组件，所以访问的区域和权限更多，相当于进入了内部进行操作
        - demo03
