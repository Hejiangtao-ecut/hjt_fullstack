# 表格
1. 语义化标签
    - thead 表头
        - caption 标题
    - tbody 表格内同
    - tfoot 表格底部内容
    - tr 行
    - td 行的每一项
2. 内容操控
    - td 内文本框使用text-align 和 vertical-align 进行文本水平垂直方向操作
    - table 可以添加bgc来操控背景等
    - empty-cells: hide; 空的单元格进行隐藏
3. 表格间距
    - border-spacing: 0px; 控制各个单元格之间的间距
4. after和before分别在元素的后面和前面追加元素
